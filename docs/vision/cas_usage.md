# Cas d'usage -- DATIAOWNP

## 1. Santé et bien-être

Données de sommeil, activité, fréquence cardiaque → IA médicales
(statistiques anonymes uniquement).

## 2. Comportements numériques

Temps d'écran, navigation anonymisée, catégories d'app usage →
optimisation IA mobile.

## 3. Transport et mobilité

Données GPS floutées → IA d'optimisation urbaine.

## 4. Consommation

Achats anonymisés → IA retail / pricing.

## 5. Recherche académique

Envoi de micro-datasets à des laboratoires via le protocole.

Chaque usage → rémunération automatique.
