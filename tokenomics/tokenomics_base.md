# Tokenomics (premier jet) -- DATIAOWNP

## 1. Token : DATIA

Type : utilitaire + gouvernance légère.

## 2. Usages du token

-   paiement d'accès aux données
-   staking pour sécuriser le protocole
-   reward utilisateur
-   frais de réseau

## 3. Distribution (V0)

-   40% reward utilisateurs
-   20% équipe & croissance
-   20% investisseurs
-   20% trésorerie et liquidité

## 4. Mécanismes

-   burn partiel sur chaque utilisation
-   halving des rewards tous les 3 ans
